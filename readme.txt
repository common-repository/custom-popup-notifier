== Description ==
Custom Popup Notifier allows you to display customizable popups on your WordPress site. Configure settings from the admin panel, including animations, colors, and custom CSS/JS.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/custom-popup-notifier` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the 'Settings' link in the plugin row or navigate to 'Settings' -> 'Custom Popup Notifier' to configure the plugin.

== Usage ==
1. Navigate to 'Settings' -> 'Custom Popup Notifier' to configure your popup settings.
2. Enable the popup by checking the 'Enable Popup' option.
3. Configure the popup's background color, font color, font size, open and close animations.
4. Add custom CSS and JS if needed using the provided fields under the 'Additional Settings' tab.
5. Save the settings to apply the popup to your site.

== FAQ ==

= How do I enable the popup? =
Navigate to 'Settings' -> 'Custom Popup Notifier' and check the 'Enable Popup' option, then save the settings.

= Can I customize the popup's appearance? =
Yes, you can customize the background color, font color, font size, and animations from the settings page.

= How can I add custom CSS and JS to the popup? =
Go to the 'Additional Settings' tab in the plugin settings and enter your custom CSS and JS in the respective fields.

= What animations are available for the popup? =
You can select from a variety of animations for both opening and closing the popup, including bounce, fade, slide, and more.
